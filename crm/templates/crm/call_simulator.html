🔌 Go Live with a Real Phone Number

To replace the simulator with a real phone line:

  1. Sign up at twilio.com (free, ~$1/month for a number)
  2. Buy a phone number in the Twilio console
  3. Set the voice webhook to: https://thoangai.pythonanywhere.com/twilio/voice/
  4. Install the Twilio Python package: pip install twilio
  5. Update the Django app Admin Panel with sample clients

The IVR will automatically look up callers and route them — no code changes needed.
